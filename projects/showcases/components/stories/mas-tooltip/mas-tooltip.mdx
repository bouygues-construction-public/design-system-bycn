## Overview

The `MasTooltip` is an overlay that shows the user a small description of a component that requires better understanding.

It appears when hovering over the component for a few seconds and disappears as soon as the cursor leaves the component.

## MasTooltipDirective

The `MasTooltipDirective` is a custom directive that provides tooltip behavior to any HTML element.

### Usage

To use the `MasTooltip`, you can apply the `MasTooltip` attribute to any HTML element and pass the desired inputs.
