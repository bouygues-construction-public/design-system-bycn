## Overview

The `MasAlerts` are used to communicate important information to the user and attract their attention. It is contextual to the message.

## Usage

### Basic

The `MasAlerts` are displayed by calling the showAlert method provided by the `AlertService`. 
The `MasAlerts appears at the top right of the interface above the navigation bar and in front of the main content. It disappears after 10 seconds or when the user clicks on the closed button.


```html
<button (click)="showInfoAlert()">  Info  </button> <br/>
<mas-alert> </mas-alert>
```
```javascript
constructor(private alertService: AlertService) { }

showInfoAlert(){
this.alertService.showAlert({ title:'Info', description:'description for toast', type:'info', closeBtn:'true'}) ;   
}
```
The `showAlert` takes the following parameters:

`title` (required): A string representing the title of the information alert.

`description` (optional): A string providing additional information or details for the alert. It has a default value of an empty string.

`type` (optional): A string indicating the type of alert. The default value is set to 'info'.

`closeBtn` (optional): A boolean flag specifying whether to include a close button in the alert. The default value is true.


The only mandatory parameter is the title, with optional parameters for description, type, and a close button, each having default values. This allows for flexible usage with minimal required input.

## Properties

### Types

The `MasAlerts` component has 4 different types : `info`, `success`, `warning`, `error`. 
These 4 types have their own semantic colors linked to their context of use and their meaning
