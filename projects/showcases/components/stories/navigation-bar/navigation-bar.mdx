## Usage

Navigation bar contains functionalities attached to the current product as well as the entire ecosystem.

- It supports multiple feature as links, icons buttons, search-input, actions
- It host the brand Identity on it's own.

Use navigation bar as a persistent component through all your product(s)
Because it represent our brand Identity on numerical product, navigation bar must always contain a visual link to the brand, whether to a product or to the company.
