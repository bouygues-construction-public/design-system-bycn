## Technologies

* [Tailwind CSS](https://tailwindcss.com/) - All this package is generated with Tailwind CSS, a utility-first CSS framework that can be composed to build any design, directly in your markup. Be careful, in order to use our library, you will have to prefix all CSS utility classes with `mas-`.
* [Storybook](https://storybook.js.org/) - CSS guidelines & components are exposed in a Storybook to help you understand how to consume this package.
* [PostCSS](https://postcss.org/) - Under the hood, we use PostCSS to build our library.
