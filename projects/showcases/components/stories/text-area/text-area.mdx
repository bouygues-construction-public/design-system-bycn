## Overview
Textareas allow users to input unanticipated queries or longer text content not suitable for standard text inputs

## Usage

- **Textarea Usage**
  - Use textareas when users need to input larger amounts of text, such as paragraphs, comments, or other lengthy content.
  - Use textareas in forms for fields that require longer input, such as comments sections or message boxes.
  - Textareas are suitable for scenarios where users need to provide detailed descriptions, feedback, or write-ups.

To use the `Textarea` component, it serves to incorporate user input of longer text content.
