## Overview

The `MasButton` is a custom Button component serve as navigational element and can be used independently or inline with text.
it offer a lightweight option for navigation.

## MasButton Directive

The `MasButtonDirective` is a custom directive that provides button styles and behavior to any HTML element.

### Usage

To use the `MasButtonDirective`, you can apply the `MasButton` attribute to any HTML element and pass the desired inputs.
