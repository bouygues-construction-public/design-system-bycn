## Usage

Searchbar enable people to find relevant informations by enter queries, words or sentences.

- Can introduce suggestions
- Can be used as a way of discovering content
- Can be used with filters

Place the search bar where the user expects to find it, and in proximity to the content that will be influenced by it. The search bar can be on a page, within a navigation component, or in a layer
