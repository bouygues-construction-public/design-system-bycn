## Overview

`MasTab` provides a simple way to organize and present content on different sections. Users can easily navigate between different sections or categories, optimizing space and enhancing user experience on the layout.

## Usage

- Use Tabs when you have related content that can be logically grouped into different sections or categories.
- When content has a clear hierarchy, Tabs can visually represent this structure, making it easier for users to understand and navigate.
- When the application provides simple navigation with few pages, Tabs can be used in the navigation bar to navigate between large pages and sections.
