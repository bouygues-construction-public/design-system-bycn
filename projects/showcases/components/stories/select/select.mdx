## Overview

Selects are intended for selecting options from a reduced list. The list must contain a minimum of 3 items. It is a component that allows the user to input data and provide context for their choices or journey.

## Usage

- Use select component to make a selection from a predefined list of items, and when user have to set data.
- Be careful when using select to allow the user to enter data in a form. In the case of a navigation action, select is not appropriate.
