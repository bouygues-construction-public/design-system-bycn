## Overview

The `avatar` is a representation of the user. It allows the user to personalize themselves in the interface and link them to content

- `size`: The size of the avatar. Accepts values: `'XS'`, `'S'`, `'M'`, `'L'`.
- `type`: The variant of the avatar. Accepts values: `'Image'`, `'Icon'`, `'Initial'`.
- `initial`: The initial letter or character displayed in the avatar when the type is `'Initial'`.
- `Image`: Used to display an image in the avatar. 
- `Icon`: Used to display an icon in the avatar.
