## Overview

The navigation drawer allows users to navigate through multiple destinations in a hierarchical layout that is visible throughout the application.

## Usage

The navigation drawer is a component that allows you to go further than the navigation bar. Its location and content makes it possible to accommodate a large number of destinations, as well as submenus and additional actions thanks to a header or footer. Its use is very common in mobile.
