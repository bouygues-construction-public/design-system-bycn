## Overview

- The input dropdown allows the user to select one or more items from the dropdown menu.
- Unlike Select, the Input dropdown can display an icon or an image next to the text of the list items.
- Input dropdowns can be used to activate an action, filter content or as a navigation element.

## Usage

- To allow users to select multiple items from a list.
- When the number of elements in the menu list is too large.
- When it's necessary to display additional elements with the text. (icon or image)

## Variants

### Overview

Input dropdown offers several types of menu compared to select which can only display text in the list.

- `Text only`: To show text only
- `Icon + Text`: Add a leading icon to the left of the text
- `Image + Text`: Add a image to the left of the text
- `Checkbox`: To present a list for multi-selection with a checkbox
