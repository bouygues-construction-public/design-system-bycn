## Overview
A popover is a layered component that appears above all other page content. 

The `MasPopover` component provides a flexible and customizable way to display contextual information or actions in a sleek and user-friendly manner. 
With various positioning options, styling choices, and content configurations

## MasPopoverDirective

The `MasPopoverDirective` is a custom directive that provides popover behavior to any HTML element.

### Usage

To use the `MasPopover`, you can apply the `masPopover` attribute to any HTML element and pass the desired inputs.
