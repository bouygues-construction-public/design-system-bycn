## Overview

the `Navigation item` allow the user to navigate from one page to another. These elements are grouped into a navigation group which is inside the navigation drawer.

The `navigation item` can display a single level or a sub-level with submenu with a submenu displayed below.

## Variants

The component comes in different variants.

### Type

The navigation item has different types corresponding to the navigation level and whether the item has sublevels.

### Collapsed
The Collapsed property allows you to reduce the width of the item to the leading icon only, to retract the navigation drawer on the left.
