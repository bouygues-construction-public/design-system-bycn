# Spacings

> Spacings creates a visual balance that makes the user interface easier to scan, and allow consistency on different layout.

For more detail, see our [Spacing's guideline](https://zeroheight.com/9b39bb2a0/p/01e423-spacings/b/505645)

<Source language="json" code={JSON.stringify(Array.from(spacing).map(([key, value]) => {return {token: key, value: value + 'px'}}), null, 2)} />

So, here is an example with associated code:

<table>
  <thead>
    <th>Token</th>
    <th>Value</th>
    <th>Pixels</th>
  </thead>
  <tbody>
    {Array.from(spacing).map(([key, value]) => {
      if (value) {
        return (
          <tr>
            <td>{key}</td>
            <td>{value}px</td>
            <td class="table-cell">
              <div class="item" style={{ width: value + 'px' }}></div>
            </td>
          </tr>
        );
      }
    })}
  </tbody>
</table>
