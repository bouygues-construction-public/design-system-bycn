# Aspect ratio

> Media ratios are designed to ensure consistency and visibility of media content, regardless of the surface on which it is displayed. By aligning our containers with web and image standards, we ensure availability across all digital media.

For more detail, see our [Media ratio's guideline](https://zeroheight.com/9b39bb2a0/p/7303a4-media-ratio/b/505645)

## Using [CSS Custom Properties](https://developer.mozilla.org/en-US/docs/Web/CSS/Using_CSS_custom_properties)

You can use different media ratio by applied utility class to the media container.
