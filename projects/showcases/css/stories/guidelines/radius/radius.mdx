# Radius

> Corner radius is the degree of curvature at the corners of an element, such as a button or a card. Corner radius play an important role in establishing visual hierarchy and creating a consistent visual language. Here are key principles to keep in mind when designing with corner radius.

For more details, see our [Radius' guideline](https://zeroheight.com/9b39bb2a0/p/0230b8-corner-radius/b/967aaa)

## Using [CSS Custom Properties](https://developer.mozilla.org/en-US/docs/Web/CSS/Using_CSS_custom_properties)

If you don't know CSS Custom Properties, you can see basic usage documentation [here](https://developer.mozilla.org/en-US/docs/Web/CSS/Using_CSS_custom_properties#Basic_usage). All our CSS Custom Properties for colors are listed at "Swatches" column just below:

## Using [Tailwind CSS](https://tailwindcss.com) utility classes

You can use shadows with Tailwind CSS utility classes about [Border Radius](https://tailwindcss.com/docs/border-radius). Here is the list of radius extented by BYCN IT:

<Source language="json" code={JSON.stringify(tailwind.theme.extend.borderRadius, null, 2)} />

So, here is an example with associated code:

<div class="container">
  {
    Object.keys(radius).map(key => {
      if(radius[key].value) {
        return (
            <div class='item mas-shadow-400' style={{borderRadius: radius[key].value + 'px'}}>
              <span class="text">mas-rounded-{key}</span>
            </div>
        )   
      }
    })
  }
</div>
