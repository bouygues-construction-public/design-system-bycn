# Border Width

## Using [CSS Custom Properties](https://developer.mozilla.org/en-US/docs/Web/CSS/Using_CSS_custom_properties)

If you don't know CSS Custom Properties, you can see basic usage documentation [here](https://developer.mozilla.org/en-US/docs/Web/CSS/Using_CSS_custom_properties#Basic_usage). All our CSS Custom Properties for border width are listed just below:

```
var(--mas-semantic-border-width-xs);
var(--mas-semantic-border-width-s);
var(--mas-semantic-border-width-m);
```

## Using [Tailwind CSS](https://tailwindcss.com) utility classes

You can use shadows with Tailwind CSS utility classes about [Border Radius](https://tailwindcss.com/docs/border-radius). Here is the list of radius extented by BYCN IT:

<Source language="json" code={JSON.stringify(Array.from(borderWidth).map(([key, value]) => {return {token: key, value: value}}), null, 2)} />

So, here is an example with associated code:

<div class="container">
  {
    Array.from(borderWidth).map(([key, value]) => {
      if(value) {
        return (
            <div class='item mas-shadow-400' style={{borderWidth: value + 'px'}}>
              <span class="text">{key}</span>
            </div>
        )   
      }
    })
  }
</div>
