# Spacings

> Spacings creates a visual balance that makes the user interface easier to scan, and allow consistency on different layout.

For more detail, see our [Spacing's guideline](https://zeroheight.com/9b39bb2a0/p/01e423-spacings/b/505645)

<Source language="json" code={JSON.stringify(spacing, null, 2)} />

So, here is an example with associated code:

<table>
  <thead>
    <th>Token</th>
    <th>Value</th>
    <th>Pixels</th>
  </thead>
  <tbody>
    {Object.keys(spacing).map((key) => {
      if (spacing[key].value) {
        return (
          <tr>
            <td>{`--mas-core-spacing-${key}`}</td>
            <td>{spacing[key].value}px</td>
            <td class="table-cell">
              <div class="item" style={{ width: spacing[key].value + 'px' }}></div>
            </td>
          </tr>
        );
      }
    })}
  </tbody>
</table>
