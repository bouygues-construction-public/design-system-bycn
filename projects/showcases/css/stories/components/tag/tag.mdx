# Tags

## Introduction

The `tag` element is used to categorize, label and organize adjacent items using keywords to describe them.

## Default

<span class="mas-tag mas-tag_color--orange">
<span class="mas-tag_label">Default</span>
</span>

CSS styles :

```css

  /* styles here */
}
.mas-tag_label {
  /* styles here */
}
```
