# Button

## Introduction
The `Button` element is an interactive user-activated element.

Once activated, it then performs an action on the interface, such as submitting a form, changing the state of an element on the interface, or opening a dialog box.

For more details, see our [Button guideline](https://zeroheight.com/9b39bb2a0/p/520f9f-button/b/75c0d1)
